# ReactNative-Weixin

#### 介绍
使用 React Native 开发的仿微信 App

#### 技术栈
1. React-Native
2. Socket.io
3. react-native-elements
4. react-navigation
5. redux-thunk
6. react-redux
7. aurora-imui-react-native
8. axios

#### 后台链接


#### 安装教程

1.  git clone https://gitee.com/icfyl/ReactNative-Weixin.git
2.  cd ReactNative-Weixin
3.  yarn install
4.  使用 xcode 打开 ios 工程，然后运行

#### 已完成的功能

1.  登录
2.  注册
3.  通讯录
4.  两人文字聊天
